# HUBSPOT
Biblioteca de classes .NET para integração com HUBSPOT API

-> Adicione as chaves no arquivo de configuração da sua aplicação (web.config, app.config):
<code>
  <add key="user:hapikey" value="_hapikey_do_cliente_hubspot"/>
  <add key="app:urlforms" value="https://api.hsforms.com"/>
  <add key="app:url" value="https://api.hubapi.com"/>
</code>
